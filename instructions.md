steps - 
1. connect to the CIS_ROBOTS wifi network. password will be emailed.
2. clone the repo https://github.com/AlonGil-Ad/simple_mocap
3. make sure you know the NAME or ID of your "asssets" (rigid bodies)
4. make sure the asset/s are in the tracked area (cameras) 
5. run example.py, if frame number is not advancing, you are not connected to the system. 
6. notice - the angles are given in Quaternions , you can use any python library to convert to roll,pitch,yaw
